安装Claude Code for VS Code插件

安装CC Switch

什么是CC Switch？

CC Switch是Claude Code、Codex、Gemini CLI、OpenCode 和 OpenClaw 的全方位管理工具

github开源地址：https://github.com/farion1231/cc-switch
下载地址：https://github.com/farion1231/cc-switch/releases


便携版或者安装程序都行


安装完后运行启动，点击右上角➕号


选择自己有的订阅的供应商，我订阅了Minimax，我以MiniMax为例

选择供应商后向下滑动填写api-key


点击测试模型，如果界面返回绿色的响应延迟则配置成功！
